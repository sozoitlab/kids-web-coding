# 🚀 サイトを公開しよう

ついに、あなたの作品を世界中の人に見せる時が来ました！

## 🌍 公開するってどういうこと？

### 今まで vs これから
- **今まで**: 自分のパソコンでだけ見られる
- **これから**: 世界中の誰でも見られる！

### 必要なもの
1. ✅ 完成したウェブサイト
2. ✅ GitHubアカウント
3. ✅ インターネット接続
4. ✅ ワクワクする気持ち！

## 📦 ファイルの準備

### 1. CodePenからファイルをエクスポート
1. CodePenで作品を開く
2. 「Export」ボタンをクリック(下記画像参照)
3. 「Export .zip」を選択
4. ダウンロードしたファイルを右クリックしすべて展開を選択
5. 新しくウインドウが開くので右下の展開をクリック

![codepen image](../../static/img/codepen-1.png)

### 2. ファイルの確認
ダウンロードしたZIPファイルには：
```
your-pen/
├── index.html
├── style.css
└── script.js (もしあれば)
```

## 🐙 GitHubで公開する

### ステップ1: 新しいリポジトリを作る

1. **GitHubにログイン**
   - https://github.com を開く
   - ユーザー名とパスワードでログイン

2. **新しいリポジトリを作成**
   - 右上の「+」→「New repository」をクリック
   
- ![github create new repository image](../../static/img/github-1.png)

3. **リポジトリの設定**
   ```
   Repository name: jikoshoukai
   Description: 自己紹介ホームページ
   Public: ✅ (みんなに見せる)
   Add README: ✅
   ```

- ![github create new repository image](../../static/img/github-2.png)

4. **Create repository**をクリック

### ステップ2: ファイルをアップロード

1. **「Upload files」をクリック**
   
![github upload files description image](../../static/img/github-3.png)

2. **ファイルをドラッグ＆ドロップ**
   - index.html
   - style.css
   - imagesフォルダ
   
![github upload files description image](../../static/img/github-4.png)

3. **コミットメッセージを書く**
   ```
   自己紹介ページ新規作成
   ```
   
4. **緑のボタン「Commit changes」をクリック**

### ステップ3: GitHub Pagesを有効にする

1. **Settings（設定）を開く**
   - リポジトリページの「Settings」タブをクリック
   
![github upload files description image](../../static/img/github-5.png)

2. **Pagesセクションを探す**
   - 左側のメニューから「Pages」をクリック
   
![github upload files description image](../../static/img/github-6.png)

3. **ソースを設定**
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
   - 「Save」をクリック

![github upload files description image](../../static/img/github-7.png)

4. **URLを確認**
   ```
   https://[ユーザー名].github.io/[リポジトリ名]/
   ```

:::tip 💡 ポイント
公開には数分かかることがあります。コーヒーでも飲んで待ちましょう！☕
:::

:::danger ⚠️ ページを非公開にするには？
上記3.ソースコードを設定でBranchをNoneに変更してSaveボタンを押すと非公開にできます。非公開になるまで数分かかるので待ちましょう！
:::

## 🔧 よくあるトラブル

### 1. ページが表示されない

**原因と対策：**
- ファイル名が`index.html`になっているか確認
- GitHub Pagesが有効になっているか確認
- 5分ほど待ってから再度アクセス

### 2. CSSが反映されない

**HTMLでの読み込み方を確認：**
```html
<!-- ✅ 正しい -->
<link rel="stylesheet" href="style.css">

<!-- ❌ 間違い -->
<link rel="stylesheet" href="/style.css">
<link rel="stylesheet" href="C:/Users/style.css">
```

### 3. 画像が表示されない

**パスの確認：**
```html
<!-- ✅ 正しい -->
<img src="images/cat.jpg" alt="猫">

<!-- ❌ 間違い -->
<img src="/images/cat.jpg" alt="猫">
<img src="C:/Pictures/cat.jpg" alt="猫">
```

## 📢 サイトを宣伝しよう

### 1. URLを共有
```
みんな見て！私のウェブサイトができました！
https://[ユーザー名].github.io/[リポジトリ名]/
```

### 2. QRコードを作る
上のQRコードジェネレーターを使って、簡単にQRコードを作成できます！
- URLを入力するだけで自動生成
- ダウンロードして保存

### 3. SNSで共有（保護者と相談）
- 家族のSNSで紹介してもらう
- 学校の掲示板に貼る（許可をもらって）

## 🔄 更新の方法

### サイトを更新したいとき

1. **GitHubでファイルを編集**
   - ファイルをクリック
   - 鉛筆アイコンで編集
   - 「Commit changes」で保存

2. **新しいファイルを追加**
   - 「Add file」→「Upload files」
   - ファイルをアップロード
   - コミットメッセージを書いて保存

3. **自動的に更新される**
   - 数分待つと公開サイトに反映
