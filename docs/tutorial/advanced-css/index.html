💡 デザインのコツ​

色の組み合わせ​

  • 🔵 青 + 🟢 緑 = さわやか
  • 🔴 赤 + 🟡 黄 = 元気
  • 🟣 紫 + 🔵 青 = 神秘的
  • 🟠 オレンジ + 🟡 黄 = 明るい

余白を大切に​

.good-spacing {
  padding: 20px;  /* 内側の余白 */
  margin: 15px;   /* 外側の余白 */
}

統一感を出す​

  • 同じ角の丸み（border-radius）
  • 同じ影の強さ
  • 同じアニメーション時間
